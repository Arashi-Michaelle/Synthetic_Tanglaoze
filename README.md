# 合成唐牢泽
本游戏内容基于claude生成，所有游戏内容都是虚构的，如有雷同纯属偶然。
## 下载方法
由于sb牢泽的图片占用内存较大，而且github不能上传exe文件及其压缩包，因此必须分卷压缩，所以请下载两卷唐牢泽文件及zip文件，在桌面上新建一个文件夹，保证三个文件在同一个文件夹内，点击zip格式文件即可运行。如果弹出windows保护，请无视风险继续运行。目前仅支持exe文件格式。

## 游戏内容
通过鼠标控制不同形态的牢泽，下掉根据物理引擎合成不同阶段的牢泽。
健康游戏，注意防沉迷。合成成功唐牢泽的可以找发布者领一瓶阿萨姆
